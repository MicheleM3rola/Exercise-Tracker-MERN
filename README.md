# Expense Tracker App

<img src="https://res.cloudinary.com/mike88/image/upload/v1641107298/Excercise_Tracker_bdsfer.jpg" title="Excercise Tracker" alt="Exercise tracker" width="40%"/>

The Excercise Tracker App is a fullstack application (MERN), the backend is developed with Node.js, Express and Mongo DB atlas to store the exercises data and the user data, and the UI is developed with React (class Components) and CSS3 using Context api as state managment.

The Front-End is deployed on Netlify and the Back-End is deployed on Heroku
